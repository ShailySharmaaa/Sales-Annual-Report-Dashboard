# Sales-Annual-Report-Dashboard

## Project Overview

This project analyzes the 2022 transactional data of Vrinda Store to generate actionable business insights.  
The objective was to understand customer behavior, identify sales trends, evaluate channel performance, and support strategic decision-making to increase revenue in 2023.
The analysis was performed using Microsoft Excel with Pivot Tables, Pivot Charts, and interactive dashboard techniques.

---

## Problem Statement

Vrinda Store wanted to create an annual sales report for 2022 to:
- Compare Sales and Orders performance
- Identify the highest performing month
- Analyze customer demographics (Men vs Women)
- Examine order status distribution
- Identify top 5 revenue-generating states
- Determine the highest contributing sales channel

## Tech Stack
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning  
- Data Transformation  
- Dashboard Creation  
- Business Data Analysis  

## Data Source
- Internal sales dataset of Vrinda Store (2022)
- Excel file format (.xlsx / .xltx)
- Key fields included:
  - Order ID
  - Order Date
  - Sales Amount
  - Gender
  - State
  - Order Status
  - Sales Channel
  - Quantity

## Key Features
✔ Sales vs Orders comparison using a combined chart  
✔ Monthly performance analysis  
✔ Gender-based purchase analysis  
✔ Order status breakdown (Delivered, Cancelled, Returned, etc.)  
✔ Top 5 states contributing to total revenue  
✔ Sales channel performance analysis  
✔ Interactive Excel Dashboard  

## Business Insights
- Identified peak sales months indicating seasonal demand patterns.
- Women customers contributed significantly higher revenue than men.
- Majority of orders were successfully delivered, showing operational efficiency.
- A small number of states generated a large portion of total revenue.
- One sales channel dominated overall sales contribution.

## Business Impact
- Helps optimize inventory planning based on peak demand months.
- Enables targeted marketing campaigns based on customer demographics.
- Supports geographic expansion decisions.
- Assists in improving operational efficiency by analyzing order status.
- Guides marketing budget allocation toward high-performing sales channels.

## Marketing Optimization Strategy for 2023
- Increase promotional campaigns during peak-performing months.
- Focus targeted marketing on female customer segments.
- Strengthen marketing efforts in underperforming states.
- Invest more resources into the highest-performing sales channel.
- Reduce cancellations and returns through better quality control and communication.
